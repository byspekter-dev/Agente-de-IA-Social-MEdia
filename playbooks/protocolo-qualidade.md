# Protocolo de Qualidade (anti-“balela”)

## Princípios
1. **Sem fonte, sem fato**: toda afirmação factual precisa de fonte e data.
2. **Dado + insight + ação**: nunca entregar apenas observação solta.
3. **Hipótese não é verdade**: marcar explicitamente quando faltar evidência.
4. **Recência importa**: priorizar dados dos últimos 90 dias quando possível.

## Checklist de validação
- [ ] O relatório cita fontes verificáveis.
- [ ] As fontes têm data explícita.
- [ ] As recomendações estão conectadas ao objetivo do cliente.
- [ ] Existe plano de teste (A/B ou variação controlada).
- [ ] Existe métrica-alvo para avaliar a recomendação.
