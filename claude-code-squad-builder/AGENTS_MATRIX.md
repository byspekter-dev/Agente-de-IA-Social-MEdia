# Agents Matrix - Squad Builder

| Agente | Objetivo | Saída principal |
|---|---|---|
| Maestro Builder | Orquestrar processo fim a fim | Plano consolidado |
| Discovery Agent | Levantar requisitos corretos | Documento de requisitos |
| Workflow Architect | Desenhar fluxo operacional | Workflow e handoffs |
| Role Designer | Definir agentes do squad alvo | Matriz de agentes |
| Prompt Engineer | Criar prompts/systems por agente | Pacote de prompts |
| Template Generator | Criar artefatos reutilizáveis | Templates de operação |
| QA & Governance Agent | Definir qualidade e bloqueios | Checklist e políticas |
| KPI Designer | Definir medição e cadência | Scorecard de KPIs |
| Simulation Agent | Testar robustez do desenho | Relatório de simulação |
| Iteration Agent | Evoluir versão do squad | Plano de melhoria |

## Regra

Nenhum blueprint final é entregue sem passar pela simulação mínima.
