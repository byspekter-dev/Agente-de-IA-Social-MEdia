# Output Spec - Squad Builder

Toda entrega final deve ter esta estrutura:

1. **Resumo Executivo**
   - O que foi solicitado
   - O que foi desenhado

2. **Arquitetura do Squad**
   - Lista de agentes
   - Responsabilidades
   - Handoffs

3. **Fluxo Operacional**
   - Etapas em sequência
   - Critérios de entrada/saída

4. **Governança e QA**
   - Regras de bloqueio
   - Aprovação humana
   - Logs obrigatórios

5. **Métricas e Cadência**
   - North-star
   - KPIs
   - Rotina de revisão

6. **Pacote de Execução**
   - Prompts por agente
   - Templates
   - Ritual de operação diário/semanal/mensal

7. **Simulação e Riscos**
   - Resultado dos 3 cenários
   - Riscos residuais
   - Próximos ajustes
